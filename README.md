# BoneScan-AI
BoneScan AI, yapay zeka destekli bir görüntü işleme sistemidir. Röntgen görüntülerindeki kemik kanseri belirtilerini otomatik olarak tespit ederek, erken teşhis sürecine katkıda bulunur ve doktorlara yardımcı olur. Gelişmiş YOLOv8 modeli ile eğitilen bu sistem, kemik sağlığı alanında önemli bir araçtır.
